criando:

Após selecionar o botão Adicionar um servidor, você verá um pop-up que lhe dará a opção de criar um servidor do zero ou usar um modelo. Também há um espaço na parte inferior do pop-up para preencher com um convite se quiser usar um. Para este exemplo, vamos criar um servidor usando a opção Criar o meu.
